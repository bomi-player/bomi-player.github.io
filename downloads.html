Source code

You can download source code from release page. You can also get source code by cloning git repository although the source code in git repository often may be broken and unstable.

Latest release page
https://github.com/xylosper/bomi/releases/latest
Git repository (GitHub)
https://github.com/xylosper/bomi

In order to build and install from source code, please read the wiki page.
